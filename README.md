# ProjetoTestesDados

Arquivos de dados usados em testes de integração.

## Estrutura

- `config.json`: links e configuração geral.
- `news.json`: itens informativos.
- `banners.json`: destaques visuais.
- `images/`: imagens versionadas usadas pelos documentos JSON.

O campo `schemaVersion` deve permanecer em `1` enquanto a estrutura atual for compatível.
Ao trocar uma imagem, use um novo nome de arquivo para evitar versões antigas no cache.
